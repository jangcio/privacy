Privacy Notice

Shopping Helper is a Chrome extension that helps users compare product candidates from supported shopping pages. This notice summarizes the data handled by the extension.

Data used for matching

  • Current product page URL from supported shopping domains.
  • Product title and optional product metadata such as price text, image URL, brand, and model name.
  • Candidate matching results returned by the Shopping Helper backend.

Purpose

The extension uses product page data only to detect supported product pages, request similar product candidates, show comparison cards, and open an outbound product link when the user chooses to do so.

Local storage

Shopping Helper may store the last detected product, the last candidate result, and the user's enabled/disabled setting in Chrome local storage so the toolbar popup and options page can reflect the current state.

Data not collected

  • Shopping Helper does not collect payment information.
  • Shopping Helper does not collect purchase history.
  • Shopping Helper does not read or write marketplace cookies.
  • Shopping Helper does not automatically redirect users.

Affiliate links

Some outbound links may be affiliate links. Affiliate links are opened only after the user clicks a product candidate.
